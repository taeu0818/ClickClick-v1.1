## 📕 사용 방법
* 사진 촬영을 위한 웹캠이 사전에 준비되어 있어야 합니다.
* 메인 페이지 접속 시 전체 화면으로 진행해야 합니다.
* 페이지 왼쪽 상단 카메라 사용 권한을 허용해야 합니다.
* 인쇄를 진행할 때 사진을 50mmX150mm 사이즈로 수정해야 합니다.
